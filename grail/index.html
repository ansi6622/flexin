03-json

Step 1: store JSON

  1. Open the script.js file for editing.
  2. Save the data JS Object into a key named "data". Make sure the object you are storing is the JSON string. Not the JS Object. Hint: use JSON.stringify
  3. Verify in the Chrome Dev Tools you stored a JSON String and not "[object Object]".

Step 2: retrieve JSON

  1. Do a localStorage.getItem('data') and parse it into a JS Object. Hint: Use JSON.parse
  2. Modify the title on the JS Object and store it back into "data". Hint: use JSON.stringify.

Congrats! Go to the next step!
